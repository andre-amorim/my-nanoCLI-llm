# Nano-like LLM Editor

A CLI text editor that mimics GNU Nano but includes LLM capabilities for grammar checking and proofreading.

## Features
- Nano-style keybindings (Ctrl+O, Ctrl+X)
- LLM Integration (OpenAI, Grok, Gemini)
- Grammar Check (Ctrl+G)
- Proofreading

## Installation
1. `nix develop`
2. `uv sync`
3. `python main.py test.txt`
